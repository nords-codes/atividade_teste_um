nome do projeto: Atividade_teste_um
seu funcionamento reflete para o nosso primeiro funcionamento de um back end com php  
onde aprendemos o uso include, die, echo, query, header para realizar o funcionmento de um mini site de forms onde as pesoas cadastraas podem entrar e adicionar outras pessoas 
onde no include ele inclui parte de um codigo que esta em outra pagina pode ser incluida em outra pagina
o die foi usado no if pois caso a conexão desse die que seria morrer iria informar um alerta de erro de conexão
o echo ele funciona para exibir algo no terminal para que possamos verificar 
o query serve para puxar ou guardar informações no banco de dados no codigo usamos para puxar o resultado para a pagina sql do codigo 
o header funciona para conrole de requisição no codigo usamos ele para mandar uma requisição para a pagina home
para tudo isso acontecer primeiro criamos o banco de dados sql depois fomos separando tudo organizadamente para que facilitasse nosso entendimento e organização, apos isso fomos para o index crialo normalmente apos isso fizemos o form que seria para o funcionamento geral do site apos isso começamos o php logo a cima, apos isso crimos o arquivo php home que seria para o funcionamento de cadastrar novas pessoas lançando as informações delas para o banco de dados depois criamos o connect para que poassamos connectar parte de um codigo em session_star e conectar isso em paginas com o includee por ultimo criamos a parte table para que possamos mostrar a tabela de pessoas e senhas que tem acesso ao site
